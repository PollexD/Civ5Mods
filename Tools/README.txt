Tools for Civ5 Modding
======================


scale-icons
-----------

Creates scaled icon files from the given DDS-file. Creates files for sizes 256, 128, 80, 64, 45 and 32 pixels across. To change this, modify the GIMP-script below.


Install:
	- put "civ5-scale-icons.scm" to your user's home ".gimp-2.8/scripts"-directory
	- check that scale-icons.cmd has correct path to your GIMP installation

Run:

	> scale-icons.cmd YOURIMAGEFILE.dds
